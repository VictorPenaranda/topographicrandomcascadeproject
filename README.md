# Topographic Random Cascade Project
This repository includes MATLAB's scripts of the Topographic Random Cascade (TRC) model, which may be used for modelling rainfall fields ocurring over a complex terrain.
> \[!TIP]
> Technical details about the TRC model can be reviewed on the paper titled: "[An Approach for Modeling the Orographic–Forcing Effect via Random Cascades and the Long-Term Statistics of Mexico City's Daily Precipitation](https://agupubs.onlinelibrary.wiley.com/doi/10.1029/2023JD040023)".
